# Syllabus — Intermediate SQL for Data Science

1. Relational modeling & advanced JOINs — schema design and multi-way joins
2. Aggregations, GROUP BY, ROLLUP — KPIs and cohort metrics
3. Window functions & ranking — running totals, lead/lag, percentiles
4. CTEs & modular SQL — readable multi-step transformations
5. ETL patterns & upserts — staging, upsert pipelines
6. Indexing & EXPLAIN — performance and tuning
7. JSONB & geospatial features in Postgres
8. Testing & CI for SQL (SQLFluff, pgTAP examples)
9. Capstone — end-to-end ETL, modeling, analysis, and presentation

Each module includes: SQL project file (`sql/projects/`), a Jupyter notebook (`notebooks/`) and small exercises with solutions.
